Acesso Rápido

Explore o banco em 3 passos

Sem cadastro, sem cartão. Acesse o dashboard completo com dados realistas em instantes.

1

Clone o repositório

Clone via git clone e instale as dependências com npm install.

2

Ative o modo demo

No arquivo .env, defina DEMO_MODE=true e execute npm run dev.

3

Clique “Entrar como Demo”

Acesso instantâneo ao dashboard completo com saldo, transações e todas as funcionalidades.
